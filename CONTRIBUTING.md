# Contributing

## Development Setup

1. Clone the repository
2. Install dependencies
3. Run tests
4. Submit a pull request

## Commit Messages

Follow conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `chore:` Maintenance
- `test:` Tests

## Pull Requests

1. Create a feature branch
2. Make your changes
3. Run tests locally
4. Submit PR with description
